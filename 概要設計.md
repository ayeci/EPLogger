# 太陽光発電電力監視サイトからファイルをダウンロードしてLAN内の端末のブラウザで確認できるようにしたい

## 技術スタック

python / selenium / pandas / Flask

## 0. ドライバ設定

1. テンポラリフォルダがなければ作成
2. テンポラリフォルダを保存先フォルダに設定してChromeドライバを初期化

## 1. ログイン

1行の先頭20文字を取得し、

1. ログイン画面にアクセス
2. テキストフォームのID / パスワード（両方id指定）を入力
3. ログインボタン（id指定）を押下

## 2. データダウンロード

> **前提条件：** input:hidden項目があるので注意

1. 【select×3個設定】
    1. name="outputFormat", value="太陽光発電＋蓄電池"
    2. name="aggrType", value="30分データ"
    3. name="collectDate", value="yyyy-mm"
2. submitボタン(XPath="/html/body/div/div[9]/div/form/div[3]/button[2]")クリック
3. CSVファイルがダウンロードされる

## 3. 日次・月次・年次データ洗い替え

> **前提条件：** 月次ファイル保存フォルダはflaskウェブサーバ管理下のフォルダとする

1. 【ファイルコピー＆ブラウザ表示ファイル作成処理】
    1. `ダウンロードファイル` をそのままの名前でバックアップフォルダにコピー
    2. ＜Web公開フォルダに `data.csv` が存在する？＞
       1. **yes:** iii. へ
       2. **no:** ダウンロードファイルを `data.csv` として `Web公開フォルダ` にコピーし v. へ
    3. `data.csv` の末尾1行の先頭20文字を取得し、ダウンロードファイルを検索、見つかった行の次の行～最終行まで `data.csv` にコピー
    4. `data.csv` の行数が336行になるように先頭からカット
    5. `ダウンロードファイル` を削除
2. `df = pd.read_csv('data.csv')`
3. `html_table = df.to_html()`
4. Flaskで `html_table` を表示
